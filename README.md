# 🎣 DREDGE Strategy Guide

**https://dredge.notes.fit**

> Complete fishing encyclopedia, equipment guide, area overview, quest walkthrough & FAQ

A multilingual single-page strategy guide for **DREDGE** — the Lovecraftian fishing adventure by Black Salt Games & Team17.

Fish Locations · Equipment Upgrades · Area Guides · Quest Walkthroughs · NPC Directory

---

## ✨ Features

- 🐟 **Complete Fish Encyclopedia** — every fish with catch location, time, depth, method, and pricing
- 🎛️ **Multi-filter Search** — filter by region, time of day, catch method, water depth, and fish type
- 🧬 **Aberrant Variants** — marked with purple border, separate pricing, and visual indicators
- ⚙️ **Equipment & Upgrades** — all rods, engines, trawl nets, crab pots, lights, and hull upgrades with unlock conditions
- 🗺️ **Area Overview** — 6 regions with depth ranges, descriptions, NPCs, and catchable species
- 📋 **Quest Guide** — NPC directory (expandable cards), main story timeline, side quest list
- ❓ **FAQ** — answers to common questions about gameplay mechanics
- 🌐 **Multi-language** — 中文 / English with language switcher
- 🌊 **Nautical Dark Theme** — ocean-inspired color palette with teal accents
- 📱 **Responsive** — desktop and mobile optimized

## 🗂️ Content

### Fish Encyclopedia

All fish species across 7 regions with detailed catch data:

| Filter | Options |
|---|---|
| Region | Coastal, Oceanic, Mangrove, Volcanic, Ice, Abyssal, Hadal |
| Time | Day, Night, Anytime |
| Method | Rod, Trawl Net, Crab Pot |
| Depth | Shallow (0-5m), Deep (5m+) |
| Type | Regular, Aberrant |

Each fish card expands to show specific sub-locations, catch conditions, and pricing (normal vs. aberrant).

### Equipment

Upgrade paths for all 6 equipment types:

- Fishing Rods (Coastal → Oceanic → Mangrove → Abyssal → Hadal → Volcanic → Ice)
- Engines (speed upgrades, 6 tiers)
- Trawl Nets (passive catching, 3 tiers)
- Crab Pots (automated trapping, 2 tiers)
- Lights (night visibility + panic reduction)
- Hulls (cargo capacity expansion)

### Areas

| 3 | Stellar Basin | 0-5m | Monster in the center |
| 4 | Twisted Strand | 0-5m | Mangrove maze |

### Quests

- **NPC Directory** — expandable cards with location, quest/story/shop tags, dialogue tips
- **Main Story Timeline** — step-by-step walkthrough for each relic hunt
- **Side Quests** — all optional pursuits with rewards

## 🚀 Quick Start

```bash
# Open directly in browser
open dredge-guide.html
```

No build step. No dependencies. Single HTML file — all data embedded.

## 🧠 Design

- All fish data, equipment info, NPC details, and quest steps stored as structured JavaScript objects
- i18n dictionary pattern for multi-language support with fallback to browser detection
- Expandable card UI pattern — compact browsing, expand for details
- Client-side filtering with real-time stats counter
- Dark ocean theme: deep navy background, teal accents, wave-inspired gradients
